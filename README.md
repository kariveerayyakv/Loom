# LOOM - Anonymous Student Grievance System

LOOM is a full-stack web application that allows students to anonymously submit complaints and vote on issues within an institution.

## Features
- Anonymous complaint submission
- Voting system for issues
- Separate student and admin portals
- Complaint review and resolution system

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask
- Database: MySQL

## Purpose
The project was built to encourage transparent and honest student feedback while maintaining anonymity.

## Future Improvements
- Authentication system
- Real-time notifications
- Improved dashboard analytics
